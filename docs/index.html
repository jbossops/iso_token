SABA TOKEN (SAT)


Saba Token (SAT) jest dostępny na sieci testowej Rinkeby projektu Ethereum. Token SAT jest kompatibilny ze standardem ERC20.

Aby wyśwtlić formularz zakupu tokena SAT należy w przeglądrce Firefox lub Google Chrome zinstalować dodatek Metamask oraz posiadać konto w sieci Rinkeby. Do celów demonstracyjnych poniżej udostępnino klucz prywatny to konta testowego. Konto należy zaimportować wg instrukcji: INSTRUKCJA.UWAGA! - w sieciach produkcyjnych nigdy nie należy udostępniać kluczy prywatnych.
